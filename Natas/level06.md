# Natas Level 6

## Steps
1. Opened the level page in the browser.
2. Encountered an input field requesting a secret code.
3. Viewed the page source and identified a PHP script referencing `includes/secret.inc`.
4. Accessed `secret.inc` and found the hardcoded `$secret` value.
5. Entered the extracted secret into the input field.
6. Successfully obtained the password.

## Summary
Introduces extracting a secret key from the source code.
