# Bandit Level 11

## Level Goal
The password for the next level is stored in the file `data.txt`, which contains base64 encoded data

## Commands Used
- ls
- base64

## Steps
1. Used `ls` to list files.
2. Used `base64 -d data.txt` to decode the text file encoded in base64.

## Summary
This level introduces base64 encoding and decoding.
