# Bandit Level 18

## Level Goal
There are 2 files in the homedirectory: passwords.old and passwords.new. The password for the next level is in passwords.new and is the only line that has been changed between passwords.old and passwords.

## Commands Used
- ls
- diff

## Steps
1. Used `ls` to list the files.
2. Ran `diff passwords.old passwords.new` to find the only changed line.

## Summary
This level introduces comparing changes in files.
